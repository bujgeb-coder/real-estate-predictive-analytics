# AI Driven Predictive Analytics and Decision Support System for Real Estate Market Analysis

This repository contains a working front-end prototype for a capstone project focused on forecasting housing prices, comparing regional market trends, and evaluating investment risk using historical housing and macroeconomic indicators.

## Features
- Forecast dashboard
- Market driver analysis
- Property comparison table
- Scenario simulation panel

## Tech Stack
- React
- Vite
- CSS
- Git / GitHub

## Getting Started
```bash
npm install
npm run dev
```

## Recommended Git Workflow
- `main` for stable versions
- `feature/*` branches for new work
- commit often with descriptive messages
